# student-knowledge-exchange

A skill-trading platform for students to teach and learn from each other.

Repository structure

- backend/ — Node.js server, database, and API routes
- frontend/ — Static pages and client-side JS

Quick start

Backend:

```powershell
cd backend
npm install
node server.js
```

Frontend:

- Open `frontend/index.html` in a browser, or serve it with a static server.
